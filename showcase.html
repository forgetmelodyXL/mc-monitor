双协议实时探测

智能邮件告警

服务器宕机自动发送邮件通知，支持自定义冷却时间，避免告警轰炸。可配置 SMTP 服务器，支持 QQ 邮箱、163 等。

👥

多用户权限管理

四级角色体系（超级管理员 / 管理员 / 协管 / 普通用户），支持注册开关、邮箱验证、角色隔离，保护 admin 账号安全。

🌐

公开分享页面

一键将服务器状态设为公开，生成分享链接给玩家。支持显示名称、玩家列表、在线状态，无需登录即可查看。

📊

监控指标看板

可视化展示服务器在线率、历史延迟趋势、玩家峰值等关键指标，支持按小时/天/周聚合查看。

一键部署启动

Windows 用户下载单个 EXE 即可运行，Python 开发者可直接 python launch.py 启动。内置 SQLite，零数据库配置。

FAQ
常见问题
需要什么环境才能运行？
Windows 用户只需下载 EXE 文件，双击即可启动。Python 开发者可克隆仓库后运行 python launch.py。内置 SQLite 数据库，无需额外安装数据库。
支持哪些 Minecraft 版本？
支持 Java Edition 全版本（1.7 - 最新版）和 Bedrock Edition。针对 1.12.2 等低版本的 MOTD translate 组件做了专门适配。
邮件告警如何配置？
在管理后台配置 SMTP 服务器信息，支持 QQ 邮箱、163、Gmail 等。开启后可在个人中心设置邮件告警开关和冷却时间。
是否支持公网部署？
支持。可通过 Nginx 反向代理进行公网部署，支持 HTTP 访问。首次启动会自动生成随机管理员密码，保障安全。
