# MC Server Monitor · Minecraft 服务器监控面板

一个基于 Python + Flask 的 Minecraft 服务器监控网页。**不依赖任何第三方 API**，直接用原生 TCP + Minecraft Server List Ping（SLP）协议与 MC 服务器通信。

## 🚀 快速开始（最常用方式）

1. 在本文件夹中，双击 `launch.bat`
2. 第一次运行时会自动：
   - 下载便携版 Python（约 10 MB，免安装）
   - 安装 Flask 依赖
   - 启动 Web 服务器
3. 浏览器会自动打开 `http://127.0.0.1:5000`
4. 页面加载后，点击「注册」创建第一个账号并登录

> 💡 每次要使用本工具，只需要双击 `launch.bat` 即可。

## 👤 默认管理员账号

程序启动时会**自动创建**一个管理员账号：

| 项目 | 值 |
|------|------|
| 用户名 | `admin` |
| 密码 | `admin` |

⚠️ **重要**：首次登录后，请立即进入「管理后台」 → 顶部修改自己的密码，不要一直使用默认密码！

## 📖 使用说明

### 1. 公共主页（无需登录）
- 访问 `/`（首页）即可看到所有**公开**的 MC 服务器
- 显示每台服务器的：在线状态、玩家数、版本、延迟、MOTD
- 右上角有「登录 / 注册」入口

### 2. 我的管理（登录后可用）
- **添加服务器**：填写名称、主机/IP、端口，点击添加
- **公开 / 私有切换**：每台服务器可以选择是否公开到公共主页
- **立即刷新**：点击「🔄 立即刷新」查看当前状态

### 3. 管理后台（仅管理员可用）
- **用户列表**：查看所有注册用户，重置密码、升级/降级管理员、删除用户
- **服务器列表**：编辑/删除任意用户的服务器，切换公开状态
- **为任意用户添加服务器**：管理员可以直接为某个用户添加服务器

## 🌐 部署到公网

如果你有一台公网服务器，把本项目部署上去，所有访问你 IP 的人都能看到公共主页。

### 在 Windows 上部署

1. 双击 `launch.bat`
2. 服务器默认监听 `0.0.0.0:5000`（局域网 + 公网都可达）
3. 开放防火墙端口：
   ```powershell
   netsh advfirewall firewall add rule name="MC Monitor" dir=in action=allow protocol=TCP localport=5000
   ```
4. 如果使用云服务商（阿里云/腾讯云/AWS 等），还要在控制台的「安全组」放行 5000 端口
5. 他人访问：`http://你的公网IP:5000`

### 想直接用 80 端口（不带端口号）

修改 `app.py` 中的 `app.run(host="0.0.0.0", port=5000)` 把 `port=5000` 改成 `port=80`，然后重启 `launch.bat`。然后访问 `http://你的公网IP`。

## 🧠 技术原理

- **通信协议**：Minecraft Server List Ping（SLP），通过 TCP 直接与 MC 服务器握手，不需要任何 API 密钥
- **延迟计算**：测量 TCP 连接到收到 SLP 响应的耗时
- **数据库**：SQLite 单文件 `mcmonitor.db`，存用户数据与服务器配置
- **会话管理**：Flask session + 签名 Cookie，`PERMANENT_SESSION_LIFETIME = 7 天`
- **密码安全**：PBKDF2-HMAC-SHA256 + 随机盐，迭代 20 万次
- **超时保护**：后端线程超时 + 前端 `AbortController` 双重保护，避免服务器挂死

## ❓ 常见问题

**Q: 测试服务器一直显示离线？**
> 请确认：
> 1) 服务器地址和端口正确（默认 MC 端口是 25565）
> 2) 本机网络能连通该服务器（可以用 `ping` 和 `telnet host port` 测试）
> 3) 服务器**必须是 Java 版**，基岩版使用不同协议暂不支持

**Q: 忘了自己的密码？**
> 管理员可以在「管理后台」重置任意用户的密码。如果你忘了管理员密码，删掉 `mcmonitor.db` 后重启，会重新生成 `admin/admin`。

**Q: 数据库在哪里？**
> 所有数据保存在本目录下的 `mcmonitor.db` 中，直接拷贝该文件即可完整迁移。

**Q: launch.bat 第一次卡住下载 Python？**
> 如果网络受限，可以**手动**把便携版 Python 解压到 `runtime\` 文件夹中，再运行 `launch.bat` 即可跳过敏捷下载。

**Q: 如何停止服务器？**
> 在运行 `launch.bat` 的黑色命令行窗口里按 **Ctrl+C**，然后按任意键退出。

## 🛡 安全建议

1. **把默认 admin 密码改掉** — 这是最重要的一步
2. 不建议把 `mcmonitor.db` 放到可被公开访问的目录
3. 如果部署到公网，建议配置 HTTPS + 防火墙白名单
4. 不要把服务器的公网 IP 随意告诉他人
