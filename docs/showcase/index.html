强大功能，轻量实现

所有功能均由纯 Python 实现，依赖极少，部署简单

📊

实时监控

后台定时轮询服务器状态，自动记录在线人数和延迟历史，支持自定义轮询间隔。

📈

历史图表

Chart.js 双 Y 轴折线图，直观展示在线人数和延迟变化趋势，支持时间范围切换。

🔔

掉线告警

服务器上下线自动检测，弹窗通知 + 历史告警记录，支持邮件告警推送。

👥

多用户系统

完善的用户注册登录系统，支持超级管理员/管理员/协管员/普通用户四级权限。

🎨

深色模式

支持亮色/深色主题切换，跟随系统或手动选择，所有界面完美适配暗色模式。

📱

PWA 支持

可安装到桌面或手机主屏，离线访问基础界面，体验接近原生应用。

🔒

安全防护

CSRF 令牌、Rate Limiting 限流、PBKDF2 密码哈希、HttpOnly Cookie 多重安全保障。

📦

多种部署

支持 Windows EXE 单文件、Docker、Linux 一键脚本、Python 源码等多种部署方式。

🧩

分组管理

服务器按分组归类，支持排序调整，搜索和状态筛选快速定位目标服务器。

⚡

PJAX 无刷新

页面间导航无整页刷新，顶部渐变进度条，浏览器前进/后退完美支持，体验如原生应用。
